Welcome to Acrome Smart Motor Driver Docs

The Acrome Smart Motor Driver (SMD) Documentation serves as a comprehensive guide for managing motors and sensors through a master using a developed protocol. This resource is designed to facilitate quick control over any motor in use. It provides detailed information on board specifications, including pinout and sensors. It also covers interfaces and protocols, including a Protocol-RS485 Guide and an Auto-Tuner. The documentation further provides instructions on how to use Python API and Arduino Library, supplemented with example codes. Contact information is readily available for further assistance. This documentation is a valuable resource for anyone seeking to use Acrome's Smart Motor Drivers. 


`Acrome Robotics homepage. <https://acrome.net/>`__
